

Abra el proyecto en vs code (recuerde estar en la carpeta raiz
cr.ac.ucr.lenguajes.ionic.angular.BCIC\cr\ac\ucr\ionic\angular\BCIC)
e inicie el proyecto con el comando npm run start si este no funciona
es que podrian faltar las dependencias en este caso utilice el comando
npm install

despues de haber iniciado el proyecto copie el link que salen en la consola de 
el vs code y peguelo en el buscador de su navegador
o utilice ctrl y click derecho para abrir la aplicacion en el navegador
despues dirijase al menu que se encuentra
a la izquierda del navegador  click en la opcion de "cursos"
dentro de este apartado se puede realizar el crud pertinente

en el lado inferior del navegador se encuentra un boton con el signo de +
el cual sirve para ingresar un nuevo curso ingrese los datos requeridos
y click al boton crear para crear un nuevo curso despues 
en el boton de detalles se encontraran los detalles de ese curso
en el boton que dice actualizar se podra modificar los
datos necesarios y en el boton con el signo de x se eliminaran
